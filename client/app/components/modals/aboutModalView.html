Until recently, file sharing in browser had to be done by uploading a file to a server, and then sending it from there to the desired destination.

MKStream, powered by WebRTC, can be used to create a lightning fast connection for sharing files of any kind with anyone you want. All file sharing is done peer to peer, meaning that the data never touches our servers and remains secure and private. Our database is only used to store the unique links, which are deleted once you finish using them.

MKStream was developed by four full stack software engineers: Tyler Ferrier, Malek Ascha, Kevin Van and Simon Ding. MK Stream uses Angular for the front end, Node/Express and MongoDB for the backend, and Karma and Jasmine for testing. We used PeerJS to harness WebRTC and create our peer-to-peer connection for file sharing.
